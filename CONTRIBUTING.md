# 一起補完這本圖鑑

歡迎回報錯漏、補充活動、提供照片！

## 回報資料錯誤 / 補充活動

最簡單的方式：直接在 Google Sheet 修改/新增一列（如果你有編輯權限），
網站會即時更新。沒有編輯權限的話，請透過以下方式回報：

- 在 GitHub 開 Issue（推薦，可附連結與截圖）
- 或聯絡維護者（見 README）

回報時請盡量附上：日期、活動名稱、出席聲優、樂團/角色、來源連結。

## 欄位說明

一筆活動可填的欄位（Sheet 表頭）：

- 必填：`年份`、`開始日期`、`結束日期`、`月份`、`活動名稱`、`類型`、
  `人物`、`團體／關聯`、`本體／擦邊`、`全團`、`人次`
- 選填：`地點`、`城市`、`照片`、`封面`、`購票連結`、`主辦`、`簡介`、`心得`、`來源`

細節見 `README.md`。

## 改程式

```bash
npm install
npm run dev        # 本地預覽
npm test           # 跑單元測試
npm run validate   # 檢查資料
npm run build      # 產出 dist/
```

送 PR 前請確認 `npm run validate`、`npm test`、`npm run build` 都通過
（CI 也會自動檢查）。
